# NEWS HIGHLIGHT
#### This is a web application to show news articles from various sources, 20th Feb 2019

### By **KARENZI Angela**
##  Description
News highlight is an application that will help the user list and preview news articles from various sources. According to whats previewed, the user can choose to see news articles from the selected source. Once they click on the link, the user can see information like the image, description and date of creation of the news article and can also read the full article by selecting on the read more link.

## Setup/Installations

### Prerequisites
* Python3.6
* Pip
* Virtualenv
* flask

### Cloning
* In your terminal <br>

    $ source <name of virtual environment>/bin/activate<br>
    $ git clone https://github.com/Anabella1109/News.git<br>
    $ cd News-Hughlight

### Running
 * In your terminal<br>

     $ chmod +x run.py<br>
     $ ./run.py

### Test
* To run tests
  * In your terminal<br>
  $ python3.6 source_test.py<br>
   $ python3.6 article_.py
